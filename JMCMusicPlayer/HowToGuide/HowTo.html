The JMC Music Player is an MP3 media player application designed to allow Jupiter Mining Corporation
employees to log into the music player client and play their favourite music from any PC on the JMC network.
User credentials are verified on a central server and upon successful login, granted access to the music player.

Please see the How-To Guide (Help > How-To Guide) for assistance in operating the application.

Please forward feedback and bug reports to rpieri@live.com.au.

Logging In

Logging into the Music Player

To login to the music player, on the login screen enter the server IP and fill in the input
fields under the "Login" heading, then click 'Login'.
If login is successful the music player will open.


Signing Up

To sign up for an account, on the login screen enter the server IP and fill in the input
fields under the "Sign Up" heading, then click 'Sign Up'.


Saving CSV files as...

To save a CSV file to a new file/location, go to File > Save CSV As.
A Save File dialog will appear and allow the user to select a file to overwrite or another location to save the file to.

Loading & saving playlists

Loading playlists

To load a playlist, go to File > Load Playlist.
Select an existing playlist from the Open File dialog window and click 'OK'.
The playlist will be displayed in the playlist window.


Saving playlists

To save a playlist, go to File > Save Playlist.
If the currently opened playlist is an existing playlist it will be overwritten, otherwise the Save Playlist As
dialog window will appear.


Saving playlists as...

To save a playlist to a specific location or file, go to File > Save Playlist As.
In the Save Playlist As dialog window, navigate to the directory you want to save the playlist in and click 'OK'.
The current playlist will be saved to the selected location.

Playlists

Creating playlists

To create a new playlist, go to File > New Playlist. Enter a playlist name into the New Playlist dialog window and click 'OK'.


Adding songs to a playlist

To add songs to a playlist, go to Playlist > Add Songs. Select the songs you want to add from the "Add Songs"
dialog window that appears and click 'OK'. The selected songs will be loaded into the current playlist.
If there are duplicate songs found an alert window will appear notifying the user of which songs have not been added.


Removing songs from a playlist

To remove songs from a playlist, select a song in the playlist window, then go to Playlist > Remove Song.
The selected song will be removed from the current playlist.


Clearing all songs from a playlist

To clear all songs from a playlist, go to Playlist > Clear Songs. All songs in the playlist will be removed.

Player Controls

Playing songs

To play a song, select the song from the playlist window and click the 'Play/Pause' button.
Alternatively, you can double-click on the song in the playlist.


Previous/Next songs

To return to the previous song, click the "< <" button.
To skip to the next song, click the "> >" button.
